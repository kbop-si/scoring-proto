# KBO 기록 앱

기록입력시스템
기록지에 있는 모든 정보를 담아야함
트래킹 관련 데이터는 담지 않음

---

## 기능

### v1

- **라인업 입력** — 현장 전달 방식과 동일한 순서로 입력
- **다이아몬드 현황판** — 수비 포지션 및 주자 위치 시각화
- **선수 교체** — 우측 버튼으로 교체 입력
- **타석 전체 삭제**
- **대기심 정보 입력**
- **기본 기록**
  - 볼카운트 (B/S/O)
  - 아웃
  - 안타 / 홈런 / 득점 / 도루

### v2 (예정)

- **연결동작** 기록
- **기록지 심화 표시**
- **문자중계** — 기록지와 연동한 땅표 정보 입력
- **메모 기능**
- **디플렉션** 다양한 경우의 수 정리 및 추가
- **타자 타석 방향 선택** (우타 / 좌타)
